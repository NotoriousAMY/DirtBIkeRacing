Logical Data Model


 TABLE Series 
  series_id       Primary Key
  series_name     
  series_desc     


 TABLE Event 
    event_id        Primary key
    series_id       Foreign Key
    event_name      
    event_date      
    location        
    city            
    state           


TABLE Class 
    class_id        Primary key
    class_name      
    class_class     


 TABLE Racer 
    racer_id        Primary key
    racer_name      
    racer_number    
    racer_city      
    racer_state     
    racer_bike        
  


TABLE Results 
    result_id       Primary key
    racer_id        Foreign Key
    event_id        Foreign Key
    class_id        Foreign Key
    place              
    place_numeric              
    laps            
    finish_time        
    points_earned   
    brand           
    number          



Relationships
	All fact to dimension relationships are Many to One (M:1)
	One Series has Many Events
	One Event  has Many Results
	One Class  has Many Results
	One Racer  has Many Results
